# Research Assistant

## What it does

Turns a single focused question into a sourced, cited brief for a real decision. It plans distinct sub-questions, gathers sources with web search and fetch, prefers primary and authoritative sources, and verifies each material claim adversarially by looking for sources that disagree or qualify it. You get a Markdown brief that leads with the bottom line, then lists key findings with cited URLs, evidence and disagreement, confidence and gaps, sources with dates, and recommended next steps.

## Setup

Customize `agent/instructions.md` with the questions, source standards, and decisions your project cares about.
