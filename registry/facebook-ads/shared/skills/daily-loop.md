---
name: daily-loop
description: Review yesterday's Facebook Ads performance and prepare optimization recommendations.
---

1. Load recent campaign insights.
2. Compare spend, conversions, and CPA against the previous period.
3. Flag campaigns with rising CPA, falling conversions, or underspend.
4. Recommend budget, targeting, or creative follow-up for an operator to approve.
5. If prior daily summaries are available in the thread or repo, mention the broader trend briefly.
6. Return a concise operator-ready summary and state that no campaign changes were made.
