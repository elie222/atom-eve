# Meeting Notes

## What it does

Reads a Fireflies transcript and produces structured meeting notes: summary, key decisions, action items with owners, risks or open questions, and follow-up drafts for email, tasks, or team channels. It uses transcript text and Fireflies metadata as source material, then leaves sharing and task creation to a human.

## Setup

Create a Fireflies API key with read access to transcripts. Add any standing note format, team naming conventions, follow-up destinations, and decision/action-item rules to `agent/instructions.md`.
