# Meeting Notes

## What it does

Turns a Fireflies transcript into operator-ready notes you can act on.

From the transcript it produces:
- a meeting summary
- key decisions, each with the speaker context behind it
- action items with owners and due dates only when the transcript makes them clear
- open questions and risks
- follow-up drafts for email, tasks, or team channels

It preserves uncertainty rather than inventing owners, dates, or commitments, and leaves sharing and task creation to you.

## Setup

Create a Fireflies API key with read access to transcripts.

Add any standing note format, team naming conventions, follow-up destinations, and decision or action-item rules to `agent/instructions.md`.
