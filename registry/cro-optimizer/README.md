# CRO Optimizer

## What it does

Hands you a ranked backlog of A/B tests for the conversion blockers on your landing pages.

Opens each landing page in a real browser, captures screenshots, and evaluates conversion heuristics:
- above-the-fold clarity and value proposition
- primary CTA prominence and copy
- friction in the conversion path
- trust and social proof
- visual hierarchy, readability, page weight, and mobile behavior

It applies marketing-psychology to explain why each change should move behavior, then returns ranked A/B test ideas, each with a falsifiable hypothesis, expected impact, effort, primary metric, and a measurement note.

## Setup

Set the pages to audit, your funnel, audience, offer, and conversion goals in `agent/instructions.md`.
