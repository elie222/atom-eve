# CRO Optimizer

## What it does

Opens your configured landing pages in a real browser, captures screenshots, and evaluates conversion heuristics: above-the-fold clarity, value proposition and message hierarchy, primary CTA prominence and copy, friction in the conversion path, trust and social proof, visual hierarchy and readability, page weight, and mobile behavior. It applies the marketing-psychology skill to explain why each change should move behavior, then returns a Markdown report with ranked A/B test ideas, each with a falsifiable hypothesis, expected impact, effort, primary metric, and a measurement note. It compares against prior runs when history exists.

## Setup

Pass the pages to audit in the prompt, or point the agent at local config that lists them. Tune `agent/instructions.md` for your funnel, audience, offer, and conversion goals.
