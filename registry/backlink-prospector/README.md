# Backlink Prospector

## What it does

Turns your competitors' backlinks into a vetted outreach queue you can work from a spreadsheet.

Finds domains that link to your competitors but not to you, then checks each prospect for context, fit, contact paths, and obvious disqualifiers. You get a CSV with:
- the referring domain and competitor evidence
- prospect notes and fit assessment
- a contact URL or email
- a status and next action

It verifies contact forms and email addresses but never submits forms or sends outreach.

## Setup

DataForSEO API credentials with Backlinks API access. Set your domain, competitor domains, daily prospect target, and any exclusion rules in `agent/instructions.md`.
