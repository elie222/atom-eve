Audit the target website as a developer-focused QA pass.

Default checks:

- Page load and obvious rendering failures
- Navigation and primary calls to action
- Mobile viewport usability
- Title, meta description, canonical URL, and heading structure
- Broken or suspicious links
- Accessibility basics: labels, alt text, contrast risks, keyboard-obvious controls
- Copy clarity and trust signals
- Console/network errors when browser tooling exposes them

Report format:

- Summary
- Critical findings
- Major findings
- Minor findings
- Screenshots and evidence
- Recommended next actions

Customize this file after installation with your product-specific flows, user personas, routes, and acceptance criteria.
