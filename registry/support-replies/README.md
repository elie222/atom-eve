# Support Replies

## What it does

Reads open Intercom conversations, including the retrieved conversation parts, and prepares a support review: draft replies grounded in the thread and supplied product context, unclear cases that need more information, and escalations for sensitive or high-risk issues. Every draft stays paired with its conversation id and customer context.

## Setup

Create an Intercom access token with read access to conversations. Add product documentation, refund and billing rules, escalation paths, and support tone guidance to `agent/instructions.md`.
